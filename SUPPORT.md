# Support

## Questions and general help

Use [GitHub Discussions](../../discussions) for:

- Setup and configuration questions
- "How do I…" questions
- Sharing feedback or ideas

## Bug reports

Open a [GitHub Issue](../../issues/new/choose) using the **Bug report** template.

Before filing, search existing issues to avoid duplicates.

## Security vulnerabilities

Do **not** open a public issue. Follow the process in [SECURITY.md](SECURITY.md).

## Feature requests

Open a [GitHub Issue](../../issues/new/choose) using the **Feature request** template.

## Out of scope for this repo

- Support for your specific self-hosted infrastructure or cloud setup
- General financial advice
- Questions about third-party services (Alpha Vantage, OpenAI, etc.) — contact those providers directly
