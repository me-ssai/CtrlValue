## [1.0.0] - 2026-04-23

### Added

- Accounts & Transactions — balance tracking, categorisation, CSV/OFX/QIF import
- Investments — positions, price history, valuations via Alpha Vantage, Yahoo Finance, CoinGecko, and metals APIs
- Loans — amortisation schedules, rate history, repayment tracking
- Budgets — monthly envelope budgeting with actuals vs. targets
- Net Worth Dashboard — charts across all assets and liabilities
- Multi-workspace support — multiple financial profiles under one account
- AI Agent — finance assistant powered by OpenAI or Anthropic (optional)
- Account security — lockout protection after failed login attempts
- Audit log — full record of all changes and auth events
- Docker Compose setup for self-hosted deployment
- ASP.NET Core 8 backend with Clean Architecture
- Angular 19 frontend with Angular Material
- PostgreSQL 16 with Entity Framework Core migrations
- Custom JWT auth — httpOnly cookies and refresh tokens
- BUSL-1.1 license with commercial use terms
